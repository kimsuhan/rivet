# 현재 제품

> 상태: 현행
> 적용 범위: 사용자가 이용할 수 있는 Rivet 제품 기능
> 정본 책임: 현재 제공 기능과 기능 간 경계
> 갱신 조건: 사용자에게 보이는 기능의 추가·제거 또는 책임 변경

## 제품 역할

Rivet은 팀이 업무 내용을 이슈로 관리하고, 실제 실행을 팀 작업으로 나누어 담당·전달·완료하는
협업 도구다. 프로젝트는 참여 팀과 이슈를 묶고, 각 팀은 자신의 멤버와 워크플로를 관리한다.

## 현재 제공 기능

### 계정과 워크스페이스

- 이메일 가입, 이메일 확인, 로그인·로그아웃과 비밀번호 재설정
- 워크스페이스 생성과 현재 워크스페이스 선택
- 관리자와 일반 멤버 역할, 멤버 초대·재발송·취소·비활성화
- 사용자 표시 이름과 프로필 이미지 관리

### 팀과 워크플로

- 팀 생성·수정·보관, 멤버 추가·제외와 복수 팀장 지정
- 팀별 워크플로 상태의 생성·수정·순서·기본값·비활성화
- 시스템 범주 `BACKLOG`, `UNSTARTED`, `STARTED`, `COMPLETED`, `CANCELED`
- 워크스페이스 공용 라벨의 생성·수정·보관

### 프로젝트와 업무 실행

- 프로젝트 생성·수정·보관·휴지통, 리드·기간·로고 관리
- 프로젝트별 실제 참여 팀 구성
- 이슈 생성·조회·검색·수정·휴지통과 우선순위·라벨 관리
- 참여 팀별 팀 작업 시작, 담당자·워크플로 상태 변경과 작업 완료
- 다른 참여 팀으로 작업 전달, 후속 전달과 전체 활동 이력
- 팀 작업별 배포 조건, 함께 배포할 작업과 배포 완료 처리

### 협업과 개인 업무

- 이슈 설명, 댓글, 멘션, 첨부파일과 활동 타임라인
- 전역 이슈, 내 작업, 팀별 목록·보드와 프로젝트 상세
- 다중 값 필터, 담당자 필터, 다중 정렬, 표시 필드·2단계 그룹 구성과 개인 저장된 보기
- 인앱 알림, 읽음 처리와 사용자가 활성화한 Web Push

### 관리와 데이터 이동

- 워크스페이스 공용 이슈 템플릿
- CSV 가져오기와 이슈·프로젝트 CSV 내보내기
- 이슈·프로젝트 휴지통 복원과 보존 기간 만료 후 정리
- 제품 피드백 제출과 관리자 상태 관리
- API·워커 상태 확인, 제품 이벤트와 운영 관측
