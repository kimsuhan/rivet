# Owner 문서

이 폴더는 Rivet 제품 소유자가 직접 읽고 사용하는 개인 작업 문서를 보관한다.

- 제품 최종 사용자용 도움말이 아니다.
- Codex가 자동으로 읽고 따라야 하는 `AGENTS.md`, 개발 지침 또는 skill이 아니다.
- 사용자가 내용을 검토한 뒤 필요한 프롬프트를 직접 복사해 작업을 시작한다.
- 프로젝트 정본과 개발 기준은 계속 `docs/planning`, `docs/architecture`, `docs/delivery`, `docs/development`에서 관리한다.
- 이 폴더는 `docs/index.md`의 제품·기술 문서 색인에 포함하지 않는다.

현재 프롬프트 모음은 [`prompt/alpha/`](./prompt/alpha/)에서 확인한다.
