# 개인 메모

> 사용자가 나중에 확인하기 위해 직접 남기는 메모입니다.
> Codex 작업 지침이나 제품 요구사항의 정본으로 사용하지 않습니다.

## 보기

- 필터에서 같은 항목의 값을 여러 개 선택할 수 있도록 개선
  - 현재 예: 상태 필터에서 `완료` 하나만 선택 가능
  - 원하는 예: `진행 중`, `완료`를 동시에 선택
  - 상태뿐 아니라 여러 값을 가질 수 있는 다른 필터에도 같은 방식 고려
- 보기별 표시 필드 커스텀
  - 보고 싶은 필드만 표시하고 나머지는 숨기기
  - 예: `최근 수정일` 대신 `생성일`을 표시하는 보기
  - 보기마다 선택한 필드 구성을 따로 저장
- 보기별 그룹화
  - 메인 그룹과 서브 그룹을 각각 지정
  - 예: 담당자를 메인 그룹으로 두고, 각 담당자 안에서 상태별로 다시 그룹화
  - 각 그룹의 항목 개수를 표시하고 그룹을 접거나 펼칠 수 있도록 구성
  - 보기마다 선택한 그룹 구성을 따로 저장
